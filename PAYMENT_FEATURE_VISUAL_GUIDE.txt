🔄 COMPLETE USER FLOW:
======================

1. USER BOOKS COURT
   └─> exclusive_reservations table
       └─> payment_status = 'unpaid'

2. USER SEES "PAY NOW" BUTTON
   └─> dashboard.php shows button

3. USER CLICKS "PAY NOW"
   └─> redirected to payment-info.php?reservation_id=X

4. USER SEES PAYMENT DETAILS
   └─> reads from payment_settings table
       ├─> GCash: 09123456789
       └─> Bank: BDO 1234567890

5. USER MAKES ACTUAL PAYMENT
   └─> sends money via GCash/Bank (outside system)

6. USER UPLOADS RECEIPT
   └─> file saved to uploads/receipts/receipt_1234567_abc.jpg
   └─> payments table: proof_image = 'uploads/receipts/...'
   └─> payments table: payment_status = 'pending_verification'

7. ADMIN REVIEWS RECEIPT
   └─> admin/payments.php
       └─> sees thumbnail of receipt
       └─> clicks to view full image

8. ADMIN VERIFIES OR REJECTS
   └─> clicks "Verify Paid" button
       └─> payment_status = 'paid'
       └─> verified_by = admin_id
       └─> verified_at = NOW()

9. USER SEES UPDATED STATUS
   └─> dashboard.php shows status = 'paid'
   └─> "Pay Now" button disappears
   └─> ✅ Booking confirmed!
