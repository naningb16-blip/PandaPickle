========================================
TROUBLESHOOTING
========================================

If you see errors after deployment:

ERROR: "Connection timeout"
→ Check environment variables (DB_HOST, DB_PORT, etc.)
→ Verify PostgreSQL database is running on Render

ERROR: "Table doesn't exist"
→ Import database/schema.sql using TablePlus

ERROR: "Column doesn't exist"
→ Make sure you're deploying from 'main' branch (not 'master')
→ Clear build cache and redeploy

ERROR: Still seeing MySQL errors
→ Check which branch is deployed (Settings → Branch)
→ Should be 'main', not 'master'
