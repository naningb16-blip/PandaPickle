ROUND FILTER FEATURE
====================

Date: June 20, 2026

FEATURE ADDED:
--------------

Added dropdown filters above the Generated Matches table to filter by Session and Round.
Admins can now easily manage and view specific rounds or sessions.

KEY FEATURES:
-------------

1. Filter Dropdowns
   
   a) Session Filter
      - Shows all sessions that have generated matches
      - Option: "All Sessions" (default)
      - Auto-submit on change
      - Filters matches by selected session
   
   b) Round Filter
      - Shows all rounds that have been generated
      - Option: "All Rounds" (default)
      - Auto-submit on change
      - Filters matches by selected round
      - Rounds sorted in descending order (newest first)

2. Combined Filtering
   - Can filter by Session only
   - Can filter by Round only
   - Can filter by both Session AND Round
   - Filters work together (AND logic)

3. Clear Filters Button
   - Appears when any filter is active
   - One click to reset all filters
   - Returns to "All Sessions" and "All Rounds" view

4. Empty State Messages
   - Shows different messages based on context
   - With filters: "No matches found for the selected filters."
   - Without filters: "No matches generated yet. Create a session..."

TECHNICAL IMPLEMENTATION:
-------------------------

1. Query Building
   - Base query joins all necessary tables
   - Adds WHERE conditions based on filters
   - Uses prepared statements for security
   - Maintains ORDER BY for consistent sorting

2. Filter Parameters
   - filter_session: Session ID (0 = all)
   - filter_round: Round number (0 = all)
   - Passed via GET parameters in URL

3. Dynamic Dropdowns
   - Session dropdown populated from sessions with matches
   - Round dropdown populated from distinct rounds in matches table
   - Both sorted appropriately

4. Auto-Submit
   - Dropdowns use onchange="this.form.submit()"
   - Instant filtering without clicking submit button
   - Clean user experience
