### Duplicate Prevention (New)
```php
// Check for duplicates
$check = $db->prepare('SELECT COUNT(*) FROM table WHERE conditions...');
$check->execute([...]);

if ((int) $check->fetchColumn() > 0) {
    $error = 'Already exists!';
} else {
    // Create new record
}
```

### Transaction Safety (New)
```php
try {
    $db->beginTransaction();
    
    // Step 1: Create main record
    $stmt1->execute([...]);
    
    // Step 2: Create related record
    $stmt2->execute([...]);
    
    $db->commit(); // Success!
} catch (PDOException $e) {
    $db->rollBack(); // Undo everything
    $error = 'Failed. Please try again.';
}
```
