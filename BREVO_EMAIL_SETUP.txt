WHAT IS BREVO?
--------------
Brevo (formerly Sendinblue) is a professional email service.
FREE TIER: 300 emails per day (9,000/month)
Perfect for PandaPickle's booking confirmations!

STEP 1: CREATE BREVO ACCOUNT
-----------------------------
1. Go to: https://www.brevo.com/
2. Click "Sign Up Free"
3. Enter your email and create password
4. Verify your email address
5. Complete the signup process

STEP 2: GET SMTP CREDENTIALS
-----------------------------
1. Login to Brevo dashboard
2. Go to: Settings → SMTP & API
3. Click on "SMTP" tab
4. You'll see:
   - SMTP Server: smtp-relay.brevo.com
   - Port: 587 (TLS)
   - Login: (your login email)
   - Password: Click "Generate a new SMTP key"

5. **IMPORTANT**: Copy and save the SMTP key immediately!
   It will look like: xsmtpsib-a1b2c3d4e5f6g7h8-1A2B3C4D5E6F7G8H

STEP 3: CONFIGURE SENDER EMAIL
-------------------------------
1. In Brevo dashboard, go to: Senders & IP
2. Click "Add a Sender"
3. Enter your sender details:
   - Email: noreply@yourdomain.com (or use your email)
   - Name: PandaPickle
4. Verify the sender email (check your inbox for verification link)

STEP 4: LOCAL XAMPP CONFIGURATION
----------------------------------
1. Open File Explorer
2. Go to: C:\xampp\htdocs\PandaPickle\
3. Create file: `.env` (note the dot at the start)
4. Add these lines:

```
BREVO_SMTP_USER=your-brevo-smtp-user
BREVO_SMTP_PASS=your-brevo-smtp-password-key-here
BREVO_FROM_EMAIL=your-email@gmail.com
```

⚠️ NOTE: Use the actual credentials from your Brevo dashboard
          (See .env file for the actual values)

5. Save the file

STEP 5: LOAD ENVIRONMENT VARIABLES
-----------------------------------
Create file: C:\xampp\htdocs\PandaPickle\config\load_env.php

```php
<?php
// Load environment variables from .env file
if (file_exists(__DIR__ . '/../.env')) {
    $lines = file(__DIR__ . '/../.env', FILE_IGNORE_NEW_LINES | FILE_SKIP_EMPTY_LINES);
    foreach ($lines as $line) {
        if (strpos(trim($line), '#') === 0) continue;
        list($key, $value) = explode('=', $line, 2);
        putenv(trim($key) . '=' . trim($value));
    }
}
```

Then add to config/db.php at the very top:
```php
<?php
require_once __DIR__ . '/load_env.php';
```

STEP 6: RENDER DEPLOYMENT CONFIGURATION
----------------------------------------
1. Go to Render Dashboard: https://dashboard.render.com/
2. Select your PandaPickle service
3. Go to: Environment → Environment Variables
4. Add these variables:
   
   Key: BREVO_SMTP_USER
   Value: [Your Brevo SMTP user from dashboard]
   
   Key: BREVO_SMTP_PASS
   Value: [Your Brevo SMTP API key]
   
   Key: BREVO_FROM_EMAIL
   Value: [Your verified sender email]

⚠️ NOTE: Get actual values from your .env file or Brevo dashboard

5. Click "Save Changes"
6. Render will automatically redeploy

TROUBLESHOOTING:
----------------
❌ "Brevo credentials not set"
   → Check .env file exists and has correct variables
   → Verify config/load_env.php is created
   → Restart Apache after adding .env

❌ "SMTP authentication failed"
   → Double-check SMTP key is correct (no spaces)
   → Verify SMTP user email is correct
   → Generate a new SMTP key in Brevo dashboard

❌ Email not received
   → Check spam/junk folder
   → Verify sender email is verified in Brevo
   → Check Brevo dashboard → Statistics for delivery status
   → Ensure customer email is valid

❌ "fsockopen(): SSL operation failed"
   → Enable PHP OpenSSL extension in php.ini
   → Restart Apache

❌ Local testing not working
   → Use your actual email as test customer
   → Check XAMPP error logs
   → Verify .env file is being loaded
