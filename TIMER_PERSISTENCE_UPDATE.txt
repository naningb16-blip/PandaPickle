========================================
PERSISTENT TIMER SYSTEM - UPDATE GUIDE
========================================

WHAT'S NEW:
-----------
Timers now persist across page reloads! Timer state is stored in the database instead of only in browser memory.

FEATURES:
---------
✅ Timer continues running after page refresh
✅ Shows accurate remaining time based on when timer was started
✅ Timer state saved in database (not just JavaScript)
✅ Automatic restoration of running timers on page load
✅ Orange color warning when <5 minutes remaining
✅ Audio alert when time is up

HOW IT WORKS:
-------------
1. Admin clicks "Start" button → Saves timestamp to database
2. Timer counts down in JavaScript
3. Page refresh → Calculates elapsed time from database
4. Timer continues from where it left off
5. Admin clicks "Stop" → Updates database status

DATABASE CHANGES:
-----------------
Two new columns added to exclusive_reservations table:

1. timer_started_at (TIMESTAMP NULL)
   - Stores when the timer was started
   - NULL if timer never started

2. timer_status (VARCHAR)
   - Values: 'not_started', 'running', 'stopped', 'completed'
   - Tracks current timer state

TIMER WORKFLOW:
---------------

1. PENDING RESERVATION:
   - No timer available
   - Admin must approve first

2. APPROVED RESERVATION:
   - "Start" button appears
   - Click to start timer
   - Timer status: 'not_started' → 'running'
   - Timestamp saved to database

3. TIMER RUNNING:
   - Countdown displays: HH:MM:SS
   - "Stop" button appears
   - Refreshing page: Timer continues from correct time
   - <5 minutes: Timer turns orange
   - Time's up: Shows "TIME'S UP!" in red + audio alert

4. TIMER STOPPED:
   - Click "Stop" to pause
   - Timer status: 'running' → 'stopped'
   - Can restart if needed

5. MARK AS COMPLETED:
   - Click "✓ Complete" button
   - Reservation status: 'approved' → 'completed'
   - Timer status: → 'completed'

TECHNICAL DETAILS:
------------------

DATABASE QUERY:
- EXTRACT(EPOCH FROM (NOW() - timer_started_at)) calculates elapsed seconds
- Remaining time = Total time - Elapsed time

JAVASCRIPT:
- Checks data-timer-status on page load
- If 'running', starts countdown from remaining time
- Updates every second
- Clears interval when time expires

FORM ACTIONS:
- Start Timer: POST with action=start_timer
- Stop Timer: POST with action=stop_timer  
- Complete: POST with action=completed
