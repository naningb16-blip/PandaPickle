- ✅ Game numbers show play order

### MATCH GENERATION LOGIC

**Step 1: Process Friend Groups**
- Teams with "Play with Friends" and same friend_group name are paired
- Example: "Birthday Party" team 1 vs "Birthday Party" team 2
- These get game numbers starting from 1

**Step 2: Handle Odd Teams**
- If a friend group has odd number of teams, the remaining team joins random pool

**Step 3: Match Random Teams**
- All "Random Match" teams are randomly paired
- Includes any odd teams from friend groups
- Continue game numbering sequentially

**Step 4: Assign Game Numbers**
- Friend group matches: Game 1, 2, 3...
- Random matches: continue from where friend groups ended
- Sequential order helps admin schedule courts

### DATABASE SCHEMA

**Table: open_play_registrations**
- match_preference ENUM('random', 'friends') DEFAULT 'random'
- friend_group VARCHAR(100) DEFAULT NULL

**Table: open_play_matches**
- game_number INT DEFAULT NULL

### EXAMPLES

**Example 1: Pure Random**
- 8 teams: all "Random Match"
- Result: 4 matches (Game 1, 2, 3, 4)
- All randomly paired
- Friend Groups Schedule: Empty (no friend groups)

**Example 2: Pure Friend Group**
- 6 teams: all "Birthday Party" group
- Result: 3 matches (Game 1, 2, 3)
- Only Birthday Party vs Birthday Party
- Friend Groups Schedule:
  * Birthday Party | 6 teams | Game 1-3 | 3 matches

**Example 3: Mixed**
- 4 teams: "Birthday Party" group
- 4 teams: "Company Outing" group  
- 6 teams: "Random Match"
- Result:
  * Game 1, 2: Birthday Party matches
  * Game 3, 4: Company Outing matches
  * Game 5, 6, 7: Random matches
  * Total: 7 matches
- Friend Groups Schedule:
  * Birthday Party | 4 teams | Game 1-2 | 2 matches
  * Company Outing | 4 teams | Game 3-4 | 2 matches

**Example 4: Odd Friend Group**
- 5 teams: "Birthday Party" group
- 7 teams: "Random Match"
- Result:
  * Game 1, 2: Birthday Party matches (4 teams)
  * Game 3, 4, 5, 6: Random matches (7 random + 1 odd Birthday Party = 8 teams)
  * Total: 6 matches
- Friend Groups Schedule:
  * Birthday Party | 5 teams | Game 1-2 (1 team joined random pool) | 2 matches
